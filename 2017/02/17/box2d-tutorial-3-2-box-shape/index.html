矩形

b2BoxDef

使用基类b2BoxDef创建一个矩形形状，并且设置其大小、密度、弹性、摩擦力等属性。

  • b2BoxDef
    • 继承于b2ShapeDef
    • type为e_ boxShape
    • extents来表示区域值

备注：如果将密度设置为0或者null，那么该形状是静止，即不可被移动的地面或者墙体等。
